github环境配置：

1.注册&创建仓库(名称)：

打开github仓库地址：

https://github.com/login

如果有账号直接登录，如果没有点击 create a account；

注册时需要关注您的用户名，英文字母数字组成，这个用户名也是你后续创建域名访问博客的二级域名。

创建完成用户后，可以回到个人主页，并在主页中寻找Create Repository来进行项目创建。

为后续github博客访问和域名映射顺利完成，这里我们注册的项目名称需要满足如下的要求：

{你的用户名}.github.io

比如我的用户名是sid2656，则我创建的项目名称为：sid2656.github.io

其余保持默认即可，也可以填写一下描述信息。
