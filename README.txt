
基础环境：
1.安装nodejs：https://nodejs.org/zh-cn
2.配置nodejs地址仓库：
	阿里云：npm config set registry http://registry.npmmirror.com
	原始仓库：npm config set registry http://registry.npmjs.org
3.安装git：https://git-scm.com/
4.安装Hexo：
	打开终端或者bash窗口执行：
	sudo npm install hexo-cli -g
5.初始化blog文件夹：
	hexo init blog // blog 为创建的目录，你可以自己定义



博客环境：
1. 进入到hexo项目的根目录，重命名source为source-old
2. git clone https://gitee.com/sid2656/hexo-docs-source.git source
3. 编写各个文件：
    初始化分类和标签参考原始ayer的git文档
4. 根目录hexo g 生成静态
